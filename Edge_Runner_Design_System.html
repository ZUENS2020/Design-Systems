Mirror's Edge translated into interface discipline

Speed comes from subtraction.

Edge Runner treats whitespace as structural force. Surfaces stay pale, outlines stay sharp, and the red accent appears only when something needs immediate action, directional clarity, or narrative urgency.

Accent policy One decisive signal Runner Red is reserved for primary motion, key CTA, and active route state.
Density Wide negative space Layout breathes first; chrome appears only when it clarifies direction.
Theme set Light / Dark The dark mode keeps the same velocity by flipping semantic roles only.

Foundations

Foundation tokens stay almost architectural: chalk surfaces, graphite lines, and one runner-red cut. Typography uses a condensed techno-grotesk so even large headings feel directional rather than decorative.

Color roles

Use bg-0 for open field, bg-2 for edges, accent for route-critical signals.

Type scale

Display uses 64 to 96 for cinematic command.

Body sits at 16 for clean scanning while running through information.

Mono labels are reserved for coordinates, specs, and overlays.

Spatial rhythm

Sections open on --sp-7 or larger. Components snap to --sp-3 through --sp-5.

The system should feel like long jumps between safe platforms, not crowded dashboards.

Components

Components are treated like route markers. They do not ornament the page. They expose direction, state, and urgency with as little gesture as possible.

Command buttons

Primary actions are red, secondary actions stay skeletal. Nothing else competes for attention.

Status panels

Checkpoint Live feed

Panels present immediate route facts. Borders should feel like cut lines, not soft containers.

Navigation chips

North Span Maintenance Roof Drop Zone

Labels are short, geographic, and action-oriented. Avoid abstract naming.

Motion

Motion in Edge Runner should feel like momentum transfer. The interface accelerates with clean starts and abrupt landings. Never wobble, bounce, or bloom.

Entry

Panels slide on a single axis with a fast ease-out. The move should feel like stepping into frame, not materializing from nowhere.

State change

Use border-color and accent reveals first. Large transforms are only for page-level shifts or route changes.

Timing

Fast interactions land around --dur-fast; section transitions use --dur-base. If motion slows cognition, cut it.

Iconography

Icons are wayfinding marks, not illustrations. Shapes should feel sliced from tape, scaffolding, and route paint.

Voice & Tone

The voice is tactical, concise, and physically oriented. It sounds like route guidance, not brand theater.

Principle
